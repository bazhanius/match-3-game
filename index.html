Rules

Правила

Place three or more tiles of the same color so that they are adjacent to each other vertically or horizontally. Расположите три или более одинаковых плиток в ряд по горизонтали или вертикали.

The more tiles you connect, the more points you get. Points extend the timer, which is 2 minutes. Чем больше плиток в ряду, тем больше очков. Очки продлевают таймер времени, который равен 2 минуты.

The game ends when there are no possible moves left or the timer runs out. Игра закончится, когда не останется возможных ходов или истечёт время.

Control buttons

Кнопки управления

Click the "New Game" button to start a new game. Нажмите кнопку «Новая игра», чтобы начать новую игру.

Click the "Find Move" button to highlight one random available move. The badge in the upper right corner shows the number of all possible moves at the moment. Нажмите кнопку «Найти ход», чтобы выделить один случайный доступный ход. Значок в правом верхнем углу показывает количество всех возможных ходов на данный момент.

Select a tile and click the "Delete Color" button to remove all tiles of the same color. For every 1000 points you will receive 1 booster. Выберите плитку и нажмите кнопку «Удалить цвет», чтобы удалить все плитки одного цвета. За каждые 1000 очков вы получите 1 усилитель.

Select a tile and click the "Delete Area" button to delete the selected tile and all tiles around it. For every 500 points you will receive 1 booster. Выберите плитку и нажмите кнопку «Удалить область», чтобы удалить выбранную плитку и все плитки вокруг нее. За каждые 500 очков вы получите 1 усилитель.

Select a tile and click the "Any Color" button to replace the selected tile with a "Joker Tile" that matches tiles of any color. For every 250 points you will receive 1 booster. Выберите плитку и нажмите кнопку «Любой цвет», чтобы заменить выбранную плитку на «Плитку-джокер», которая соответствует плиткам любого цвета. За каждые 250 очков вы получите 1 усилитель.
